## ⚙️ Tech Stack

### 🖥️ Frontend
- React.js
- React Router
- Axios
- Tailwind CSS

### 🧠 Backend
- Node.js
- Express.js
- MongoDB + Mongoose
- JWT Authentication

### ☁️ Deployment
- **Frontend**: Vercel  
- **Backend**: Render  
- **Database**: MongoDB Atlas

## 🎯 Features

- 🔐 **JWT-based Authentication**
- 🚘 **Browse & Filter Cars**
- 📅 **Book Cars by Date**
- 📋 **View and Manage Bookings**
- 🧑‍💼 **Admin Panel for Car Management**
- 📬 **Contact Form / Support Option**

## 🛠️ Local Setup

### 🔧 Backend

```bash
cd server
npm install
# Create a .env file
npm start
```

### 🌐 Frontend

```bash
cd client
npm install
npm start
```

## 🔐 Environment Variables

In `/server/.env`:

```env
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_jwt_secret_key
PORT=5000
```
